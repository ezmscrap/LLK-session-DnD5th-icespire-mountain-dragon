# 2026年04月 LLK例会 04/17における EINさんのおすすめアンコモンアイテム について
- 決定日: 2026/04/17

## ■ EINさんのおすすめアイテム(自薦)

### 〇アダマンティン・アーマー：アンコモン　中装鎧、重装鎧（ハイド以外）
・クリティカルヒットを通常ヒットに変える効果、正直下手な魔法の鎧よりよほど強いクリティカル出されたらACどんだけ高くても当たるからね、、、テキストが簡潔であるほど効果が強いのはTCGのセオリー通り同調がいらないのも高評価。注意点として別途鎧の代金がかかることくらいか（じゃないと500GPでプレートアーマーが買えてしまう）

### 〇ウイングド・ブーツ：：アンコモン（要同調）
・自分の移動速度で飛行できる様になる。チャージは4時間で消費は1分単位、12時間未使用でいると2時間分再チャージされる。3次元で立体移動できるアドバンテージはやはり大きい、時間制限は冒険用で戦闘中はほぼ無制限、移動速度はフライ（18メートル移動）に劣るが精神集中が必要ないためダメージに強いのが強み。

### 〇ウェポン・オブ・ウォーニング：アンコモン（要同調）
・9m以内の見方は不意を討たれずついでに寝てても起こしてくれる警報を出す武器（ウォーニングというかワーニングというか）さらにイニシアチブ判定に有利がつくこっちが本命の効果。武器を身に帯びていれば効果があるためメイン武器よりサブ武器で持ってるのがよいと思う（ダガーとかショートソード等）、同調が必要なのが玉に傷。

### 〇ガントレット・オ・オーガーパワー：アンコモン（要同調）
・いわずと知れた巨人の小手、STRを19にしてくれる。すでに18になってるヘルムはともかくステータス的に余裕がないライノスにはおすすめ、ただ同調が必要なためドラスレと合わせて枠が2枠になる。

### 〇グラブズ・オフ・シーブリー：アンコモン
・こっちもおなじみ盗賊の小手、装備すると消えるので着用してても気づかれない（この気遣いが◎）＜手先の早業の技能判定：敏捷＞と、＜鍵開け判定：敏捷＞に+5のボーナス、本職よりレンジャーやバード向きだが固定値+5はでかい、同調不要なので枠を圧迫しない。

### 〇クローク・オブ・プロテクション：アンコモン（要同調）
・ACとすべてのセービング判定に+1の効果、テキストは簡潔であるほど以下略。同調が必要だがこれとアダマンアーマーのセットは下手な同調が必要な魔法の鎧よりよっぽど優秀である。

### 〇サドル・オブ・キャバリア―：アンコモン
・この鞍にまたがっていれば、落馬せず乗騎に対する攻撃に不利が付く。騎乗戦闘するならあると便利、ファインドスティードを使えるライノス向けだが運用が面倒なので要相談か

### 〇ジャベリン・オブ・ライトニング：アンコモン
・ディアブロⅡではおなじみの電撃投槍、コマンドワードで36ｍ直線の電撃ダメージ（4ｄ6、13目標の敏捷セーブ）、もしくは投擲して命中させると通常ダメージ+4ｄ6の電撃ダメージ。サイドアームにも便利だが投擲するとセーブなしで追加ダメージな点がヤバイ、複数持って連射するとえらいことに（アンコモン投げつける都合上回収できないと財布もえらいことに）、1回使用すると夜明けまで再使用不可になるが魔法のジャベリンとしては使用可能。問題は現状敵対してる神様が雷神様なことか。

### 〇センチネル・シールド：アンコモン
・装備しているとイニシアチブ判定と＜知覚：判断力＞判定に有利が付く。盾の枠を使うが同調不要でイニシアチブに有利が付くテキストは以下略

### 〇ブーツ・オブ・ウインターランズ：アンコモン（要同調）
・装備してると冷気に耐性が付き、氷や雪の移動困難な地形を無視する。これだけでも強いがついでに防寒なしで-45℃まで、防寒装備で-73℃まで平気な無茶な靴、調子に乗って落水したり雪崩に飲まれたりすると凍死しなくても窒息死するんで注意。これからホワイトドラゴンとやりあうのにはもってこいではある。

### 〇ブルーム・オブ・フライング：アンコモン
・そのまんま空飛ぶほうき、合言葉で15ｍの移動速度で飛行ができる。ウイングブーツとの違いは速度だけでなく200㎏までの重量を運べる点にある（100㎏超えると遅くなるが）ついでに1.5㎞までの距離をリモートで飛ばせるお使い機能付き。とてもほしいが金がない（アズライト談）
　
### 〇ヘッドバンド・オブ・インテレクト：アンコモン（要同調）
・旧知力の兜、装備してると賢くなるのになぜか馬鹿に見える悲しい兜だったがヘッドバンドになって多少マシに（こいつに装備すると消えるくらいの気遣いがあれば、、、）ＩＮＴ19になる効果自体はたいていのクラスで優先度の低いＩＮＴを大幅に補強できるため優秀、ただし同調が必要。

### 〇ミスラル・アーマー：中装鎧、重装鎧（ハイド除く）
・民案知ってるミスリル製の鎧（アダマンティンもだが版権問題麺臭い）、、＜隠密：敏捷＞判定の不利と筋力不足による移動力の低下を無効にする、チェインシャツかブレストアーマーだと服の下に着れるようになる模様。隠密性に優れるが同じ値段ならアダマンアーマーが優秀過ぎる、何気にエルブンチェインは別にあったりする。

### 〇リング・オブ・ウォーターウォーキング：アンコモン
・水上歩行の指輪、同調不要で落水を防げる状況次第だが色々使い出のある効果。

### 〇リング・オブ・ウォームス：アンコモン（要同調）
・耐寒ブーツの指輪版、冷気耐性と-46℃までの防寒機能付き。ブーツとの違いは耐寒機能以外に地形効果の無効がない代わりに運搬物にも防寒機能が拡張される点が違う、効果自体は実質下位互換だがウイングブーツなどを装備していて靴の枠がない場合に、ブーツほどかさばらないのも〇

### 〇ワンド・オブ・ウォーメイジ+1：アンコモン（要同調、呪文の使い手のみ）
・呪文攻撃ロールに+1する効果と呪文攻撃時に1/2遮蔽を無視する効果がある。呪文攻撃ロール自体は初級か1，2ＬＶ魔法までしかないためあんまりだがハーフまでの遮蔽を無視する効果は敏捷抵抗の魔法（ＦＢ等）にはもってこい、この効果目当てで+1だけでも欲しい（だが金が以下略）
