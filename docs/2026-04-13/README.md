## ■ 魔法のアイテム購入 ルール

### ◆ フレーバー
- 魔法のアイテムは貴重である。そのため、売主が確実にいるとは限らない。
- ただし、すぐに換金したい売主や、在庫を一掃したい売主がいると「作成費用」より安くても幸運にも入手可能(5%～25%)である。
- 一方、適価の利益率10％～20%で換金したい作成している売主や、入手したアイテムを敵かで手放したい売主がいれば半々で入手可能(45%～65%)であり、こちらの方が確度は高い。
- 確実性を取るなら、相場より高く買い取ると言って回ることだ、そうすれば高確率で入手可能(85%～)である。
- 魔法のアイテムの購入は、仲買人となるエージェントを通して行う。
- 仲買人は、手間賃を使って、依頼主が冒険している間に、その魔法のアイテムのほしがっている人がいると話しを広げ、その魔法のアイテムを売っても良いという売主を見つけると、預かった価格で交渉を開始する。
- 一般的に預かった価格の70%～90%を売主に、残りの30%～10%が仲買人に渡され、取引は成立して依頼主に魔法のアイテムが届くのである。

### ◆ マジックアイテムなどアイテム購入ルールの基本

- 前提: 欲しいアイテムの売主を見つけてくるエージェントと友好関係になる
- 内容: 1つのアイテムの売主とつなぎを付け、購入代行してくれる
- 効果: 基本的には売主は指定価格で売ってくれるが、売主がいるかは運次第
- 判定: 1冒険に1回、運試し(1d20で必要な出目が出れば運試し成功)
- 資源: 依頼料+価格を先渡し、成功時はアイテム入手、失敗時は価格を返却
- 注意: 魔法の武具の場合、武具の基本価格を単純加算する

### ◆ 得意分野のジャンル
- 武器
- 防具
- ポーション
- 巻物
- スタッフ、ロッド、ワンド
- 指輪
- その他の魔法のアイテム

### ◆ マジックアイテムなどアイテム購入ルールの詳細
- 宣言タイミング: ゲーム終了後、レベルアップするなら行うタイミング
- 判定タイミング: レベルアップするならヒットダイスを決定するタイミング
- 判定数: 複数個を依頼している場合でも、1回の判定でまとめて決定する
- ジャンル: 基本的に、得意分野以外は一つ下のレアリティまでは探せる
- DM判断: 基本的には、エージェント一人に一品だが、DM判断で増やしても減らしても良い、一品に絞ってより上位のレアリティを求めて良いが、その後に1回休みなどのアクセントをつけることが望ましい
