# 2026年04月 LLK例会 魔法のアイテムの売却ルールについて
- 決定日: 2026/04/15

## ■ 魔法のアイテムの売却ルール

- **ザナサーの百科全書 P131準拠**

### ◆ フレーバー

- 魔法のアイテムの売却とは決してたやすい仕事ではない。詐欺師や泥棒は常にカモを探しているし、仮に正真正銘、本物の買い手が見つかったとしても、その買い手が、十分な値段を申し出てくれる保証は何処にもないのだ。
- エージェントに依頼し、1回の冒険の間(1活動時間)、25gpをかけて、魔法のアイテムを売りたいという話しを広げる。

### ◆ リソース

- 必要時間: 1活動時間
- 必要経費: 25gp
- 制限: 1キャラクターは1活動時間に1アイテムのみ、売却できる

### ◆ 魔法のアイテムの基本価格

|レアリティ|基本価格|消費型アイテムの基本価格|
|---|---|---|
|コモン|100gp|50gp|
|アンコモン|400gp|200gp|
|レア|4,000gp|2,000gp|
|ヴェリーレア|40,000gp|20,000gp|
|レジェンダリー|200,000gp|100,000gp|

### ◆ 処理

- 〈説得〉【魅力】判定

|判定結果|買値|
|---|---|
|1～10|基本価格の50%|
|11～20|基本価格の100%|
|21～|基本価格の150%|
