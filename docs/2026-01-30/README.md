- 目的: キャラクターの原型は触らず、レベルアップで変わる差分だけを書き換える

‐プルダウンメニューでキャラクターを選択し、「読み込み」

### ◆ 2. レベルを選ぶ
<img src="../images/2026-01-30-02.png" width="384" alt="新しいレベルを選ぶ">

- ふつうは未作成の、いまのレベルの次のレベルを選ぶ

### ◆ 3. 最大HPを入力
- 最大HPをルールに従って増やした値を入力
- 値を決めるのは手動です

<img src="../images/2026-01-30-03.png" width="384" alt="最大HPを入力">

### ◆ 4. 特徴を確認する
- そのレベルで、クラスにより獲得できる特徴が、保持特徴に入っているか確認する
- 入っていなければ選択候補特徴から取得
- すべての特徴から選ぶをチェックしておくと、文字通りなんでも選べるはず
- 2段目は戦技など、極めて特殊な特徴を管理するためのもの

<img src="../images/2026-01-30-04.png" width="384" alt="特徴の確認">
